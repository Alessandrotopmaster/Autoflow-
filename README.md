# AutoFlow 🚀

Plataforma que trabalha por você: configure uma vez e ela cria, agenda e publica conteúdos automaticamente — vídeos, e‑books ilustrados, minissites e páginas. Inclui a loja **BioSoul Essências** com sistema de afiliados e pagamentos integrados ao Mercado Pago.

## 📌 Estrutura simplificada
- `server.js` → lógica principal, IA e integrações
- `config.js` → valores, comissões e regras ajustáveis
- `dashboard.js` → painel do usuário e administrador
- `.env.example` → chaves e configurações pessoais
- `README.md` → este guia

✅ Poucos arquivos = menos erros, manutenção rápida e implantação ágil no Render.

## ✨ Principais recursos
- Gera 4 variações de cada produto/conteúdo
- Disparos automáticos por até 3 meses
- Afiliados com comissões de 8 % a 15 % + bônus
- Modo offline + sincronização automática
- Clonagem de páginas e criação de avatares virtuais

## 🚀 Como usar
1. Copie `.env.example` para `.env` e preencha suas credenciais
2. Ajuste valores e regras em `config.js`
3. Suba para o GitHub
4. Conecte ao Render e conclua a implantação

Feito para você crescer sua presença e vendas sem esforço.
